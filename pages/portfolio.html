Football Data Pipeline

This project is my first large-scale personal project in data engineering. I designed and built a complete ETL pipeline to collect football data from the football-data.org API, transform it using pandas, and store it in a PostgreSQL database. Through this project, I learned how to work with tools and concepts not yet covered in my academic program, such as API integration, data pipelines, SQL databases, and data processing workflows. This experience significantly strengthened my ability to learn independently and build structured, real-world projects. The next step of the project is to develop a full-stack web application (React & Node.js) to visualize and interact with the collected data.

Metro Route Calculator

This project allowed me to explore low-level programming concepts in C by modeling a metro network as a graph and computing the shortest path between two stations using Dijkstra’s algorithm. It helped me develop a deeper understanding of data structures such as adjacency lists, memory management, and how data is represented and manipulated close to the hardware level. I also worked with CSV file parsing to load real data and implemented route visualization including transfer management between lines. This project was developed in a team of two, which helped me improve my collaboration skills, share problem-solving approaches, and learn how to design solutions together.
